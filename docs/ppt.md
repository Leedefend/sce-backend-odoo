# **1. 产品定位**

智能工程管理系统是一套面向 **施工企业数字化升级** 的专业平台，覆盖：

* 工程量清单（BOQ）管理
* 专业版 WBS 工程结构
* 合同、变更、签证、结算
* 预算、台账、成本控制
* 进度、质量、安全一体化
* AI 智能驾驶舱

核心定位：

> **以 WBS 为主线，以成本为核心，以 AI 为驱动，构建可计算、可追溯、可预测的工程项目全过程数字平台。**

# **4. 专业版 WBS：平台的数字底座**

### 为什么必须有 WBS？

* 所有业务数据需要统一结构承载：清单、合同、台账、进度、质量、安全
* 项目结构必须可计算、可聚合、可追溯
* 是数字化企业的“统一语言”

### WBS 六级结构（示例）

* 单项工程
* 单位工程
* 专业工程
* 分部工程
* 分项工程
* 清单项（叶子节点）

### 特点：

* **自动从清单生成**
* 支持分部分项 + 单价措施 + 总价措施 + 规费 + 税金
* 所有业务对象（成本、合同、进度等）均绑定 WBS 节点
* 形成真正“一条线贯通全流程”的数据骨骼

# **5. 五算对比体系**

五算体系包含：

| 类型 | 用途      |
| -- | ------- |
| 概算 | 项目投资控制  |
| 预算 | 目标成本    |
| 标后 | 投标与谈判依据 |
| 合同 | 执行基准    |
| 决算 | 最终结论    |

系统特点：

* 五算数据全部绑定清单项 → 不会错账
* 自动生成五算对比图、偏差分析
* 偏差来源自动定位（量差/价差/变更/漏项）

# **6. 合同 · 变更 · 签证 · 结算**

### （1）合同管理

* 主合同
* 补充协议
* 分包合同 / 劳务合同
* 材料采购 / 租赁合同

全部按清单项拆分，支持：

* 合同期望 vs 实际执行
* 分包与主合同对比（利润空间分析）

### （2）变更与签证管理

* 工程变更
* 现场签证
* 材料调整
* 紧急抢修
* 索赔事项

每条变更自动影响合同价、预算、结算与驾驶舱。

### （3）结算管理

* 中间计量
* 竣工结算
* 审计扣减
* 结算报告自动生成（PDF）

全过程留痕，可审计可追溯。

# **7. 进度 · 质量 · 安全一体化管理**

### 进度管理

* 支持任务依赖、关键路径（CPM）
* 进度由工程量驱动，避免主观填报
* 自动生成 S 曲线（PV / EV / AC）
* 延误预警、赶工分析、资源负荷分析

### 质量管理

* 检查 / 整改 / 复查全流程闭环
* 不合格项与返工影响自动进入成本结构

### 安全管理

* 隐患 / 整改 / 复查
* 安全事件自动影响进度与成本预测

# **8. 成本台账与项目经营驾驶舱**

### 成本台账（统一模型）

* 人工
* 材料
* 机械
* 分包
* 措施
* 管理费用
* 其他费用

全部绑定：

* WBS
* 清单项

* 科目

### 项目经营驾驶舱（关键指标）

* 预算成本
* 实际成本
* 主合同 vs 分包利润

* 可视化毛利趋势
* 健康评分（0–100）
* 项目群横向对比

# **9. AI 智能决策能力**

基于结构化数据，AI 提供：

* **项目健康诊断（综合评分）**
* **进度延误预测**
* **成本超支风险预测**
* **合同风险条款分析**
* **变更签证合规性判断**
* **结算报告草稿自动生成**
* **质量与安全风险推理**

AI 的本质不是“写报告”，而是：

> **协助管理者提前看到风险，提前做决策。**
