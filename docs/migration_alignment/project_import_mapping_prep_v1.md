## Pre-Import Checks

Before importing project rows:

1. Upgrade `smart_construction_core`.
2. Export current `project.project` field list after upgrade.
3. Build a dry-run mapper from CSV columns to target fields.
4. Check uniqueness of `ID`, `OTHER_SYSTEM_ID`, `OTHER_SYSTEM_CODE`, and
   non-empty `PROJECT_CODE`.
5. Decide whether `PROJECT_CODE` can be written directly or must remain
   generated by the current sequence.
6. Prepare a company and specialty dictionary matching table.
7. Prepare a lifecycle/stage conversion table for `STATE`, `XMJD`, `DEL`, and
   `IS_COMPLETE_PROJECT`.
