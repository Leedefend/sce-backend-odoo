# 能力清单（Phase N1）

能力是“用户意图 + 系统行为”的稳定编码，
不是菜单、不是模型。

项目类
- project.edit_basic：编辑项目基础信息（名称、日期、负责人）
- project.lifecycle.transition：变更生命周期状态
- project.close：项目关闭

任务类
- task.create：创建任务/工序
- task.start：启动任务执行
- task.complete：完成任务

财务类
- finance.settlement.create：创建结算单
- finance.payment.submit：提交付款申请

合同 / BOQ / 成本
- contract.create：创建合同
- boq.edit：编辑 BOQ/清单结构
- cost.entry：录入成本/进度
