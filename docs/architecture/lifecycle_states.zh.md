# 项目生命周期状态（Lifecycle States）

本文件定义 `project.lifecycle_state` 的官方语义，
作为 Phase N1 生命周期 × 能力矩阵的基础。

- draft（立项/草稿）
  - 说明：项目前期准备与立项阶段。
  - 阶段类型：管理阶段
  - 是否允许新业务对象：允许（项目基础信息、BOQ、合同）
- in_progress（在建）
  - 说明：项目执行与交付进行中。
  - 阶段类型：执行阶段
  - 是否允许新业务对象：允许
- paused（停工）
  - 说明：临时暂停，不允许新增执行类业务。
  - 阶段类型：管理阶段
  - 是否允许新业务对象：不允许
- done（竣工）
  - 说明：执行完成，进入结算准备。
  - 阶段类型：管理阶段
  - 是否允许新业务对象：有限允许（结算/付款）
- closing（结算中）
  - 说明：结算推进阶段。
  - 阶段类型：管理阶段
  - 是否允许新业务对象：有限允许（结算/付款）
- warranty（保修期）
  - 说明：保修/质保期。
  - 阶段类型：管理阶段
  - 是否允许新业务对象：不允许
- closed（关闭）
  - 说明：项目最终关闭，不再允许变更。
  - 阶段类型：管理阶段
  - 是否允许新业务对象：不允许
