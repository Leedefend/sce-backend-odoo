# Operating Line 文档入口

本目录用于沉淀“经营主线”相关的口径与模型映射，给新同事快速建立认知：

- `operating_line_v1.md`：主线定义、对象流/金额口径流（含 Mermaid 图）。
- `model_mapping_v1.md`：模型与文件的分类表（主线/投影/支撑/可删）及关键字段依赖。
- `metrics_definitions_v1.md`：关键指标的口径表（合同额、预算、已结算、已支付、可付余额等）。

阅读顺序建议：`operating_line_v1.md` → `model_mapping_v1.md` → `metrics_definitions_v1.md`。
