# TP-08｜角色表面验收清单（v0.1）

> 用于验证“表达层一致性 + 不点炸”。每条包含预期结果与失败表现。

## demo_pm

- 项目详情页：叙事卡、软提示、状态卡可见
  预期：可见且为只读文本/字段
  失败：未显示或出现可点击入口
- 项目详情页：合同/成本/BOQ/资料仅汇总，无明细入口
  预期：无列表/表单/分析跳转
  失败：出现 action 按钮或明细列表
- 项目详情页：所有状态卡/叙事卡/软提示不可点击
  预期：无链接、无按钮、鼠标悬停无手型
  失败：出现可点击链接或跳转入口
- 交互安全：无 AccessError
  预期：全程无报错
  失败：RPC_ERROR/AccessError

## demo_cost

- 成本/BOQ 明细入口可用
  预期：BOQ 列表（tree/form）、成本台账（tree/pivot/graph）、预算（tree/form）入口存在且可用
  失败：入口消失或权限不足
- 项目详情页：汇总区块仍可见
  预期：状态卡/汇总区块显示
  失败：汇总区块被隐藏

## demo_readonly

- 仅摘要区块可见
  预期：状态卡/汇总/提示可见
  失败：出现明细入口或可跳转 action
- 明细列表不出现
  预期：One2many 明细列表不展示
  失败：出现任何明细列表或 drill-down
- 交互安全：无 AccessError
  预期：全程无报错
  失败：RPC_ERROR/AccessError

## admin

- 协作入口可见可用
  预期：发送邮件/短信/分享入口可见且可用
  失败：入口不可见或不可用
- 入口基线不回退
  预期：项目表单 button_box、三点菜单、关键页签与基线一致
  失败：入口消失或被隐藏
- 动作可达
  预期：点击入口可进入对应列表/表单/分析视图
  失败：跳转失败或权限错误
- 交互安全：无 AccessError
  预期：全程无报错
  失败：RPC_ERROR/AccessError
