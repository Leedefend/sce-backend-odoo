# Phase 1 UX 验收清单

## 升级/刷新

1. `make mod.upgrade MODULE=smart_construction_core DB_NAME=sc_demo`
2. 浏览器 Ctrl+F5 强刷
3. Console 无错误

## 侧栏入口（业务域分组）

- 左侧出现“项目中心/合同中心/费用/成本/物资/采购/结算/财务/数据中心/配置中心”等域分组
- 同一权限集下顺序稳定
- 关键入口可达：项目立项、项目台账、合同台账、成本台账

## 项目草稿表单

- 顶部状态显示“草稿”，出现“提交立项”按钮
- 保存显示为“暂存”
- 草稿编辑时隐藏“放弃变更”
