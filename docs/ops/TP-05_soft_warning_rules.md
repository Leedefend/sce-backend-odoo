# TP-05｜项目软提示（Soft Warning）规则 v1

目标：给 PM 提供“注意事项”，不阻断操作、不代表审批结论。
规则尽量少（≤5），以稳定可用为优先。

## 规则清单（v1）

| 规则编号 | 触发条件 | 提示文案 | 级别 | 备注 |
| --- | --- | --- | --- | --- |
| W-01 | 成本偏差 < 0 | 成本已超预算，建议关注成本控制措施。 | 黄 | 基于 cost_budget_gap |
| W-02 | 已结算金额 > 0 AND 已付款金额 = 0 | 存在已结算未付款情况，建议关注付款计划。 | 黄 | 结算-付款 |
| W-03 | 进度完成率 = 0 且 无进度数据标记 | 暂无进度数据，建议补充进度信息。 | 蓝 | 数据提示 |
| W-04 | 资料完备率 < 80% | 资料完备率偏低，建议补齐关键资料。 | 蓝 | 阈值可调 |
| W-05 | 合同数量 = 0 | 当前项目暂无合同信息，请确认合同是否已录入系统。 | 蓝 | 辅助提示 |

## 说明

- 软提示只用于提醒，不阻断任何操作。
- 软提示不作为审批或合规依据。
- 级别：黄为关注提示，蓝为信息提示。
- 同一时间最多展示 2 条软提示，优先展示黄色提示。
