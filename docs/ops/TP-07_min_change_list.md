# TP-07｜最小执行清单（Draft）

以下清单仅描述“要改什么”，不描述“怎么改”。总量不超过 10 条。

1. PM 视角移除合同/成本/结算等原生 action 的所有入口（包括 button_box / 菜单 / 三点菜单），避免与降维策略冲突。
2. PM 视角 BOQ 原生 action 与明细列表不可见，仅保留 BOQ 状态卡（导入状态 / 总金额 / 版本）。
3. PM 视角 WBS/工程结构原生 action 不可见，改为结构状态摘要。
4. 合同页签移除明细列表与可跳转入口，仅保留汇总字段展示。
5. 成本/进度区块移除趋势/历史/分析表达，仅保留状态级结果。
6. 工程资料区块只保留完备度/缺失数量，不提供资料明细入口。
7. 协作类入口（发邮件/短信/分享）对 PM 隐藏或移出主视图。
