# Continue / DeepSeek Devtools

This folder contains optional developer tooling for Continue + DeepSeek setup. It is intentionally separated from product code.

## Quick Start

Windows (PowerShell):

```
.\tools\continue\ps1\setup-continue-deepseek.ps1
```

macOS/Linux (bash):

```
chmod +x tools/continue/scripts/setup-continue-deepseek.sh
./tools/continue/scripts/setup-continue-deepseek.sh
```

## Notes

This folder is optional. If your team does not use Continue/DeepSeek, you can ignore it.
